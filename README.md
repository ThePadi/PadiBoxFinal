# Padibox - Prosta wersja HTML

Aplikacja webowa do zarządzania profilem zapachowym użytkownika.

## Jak uruchomić

**Wystarczy otworzyć plik `index.html` w przeglądarce!**

1. Kliknij dwukrotnie na plik `index.html`
2. Lub kliknij prawym przyciskiem → Otwórz za pomocą → Wybierz przeglądarkę (Chrome, Firefox, Edge)

## Funkcjonalności

✅ **Ekran główny** - Trzy sekcje: Profil, Mapa, Punkty
✅ **Profil** - Pełna funkcjonalność:
  - Wybór wieku
  - Wybór płci (Kobieta/Mężczyzna/Inna)
  - Wpisanie miasta
  - Wybór ulubionych zapachów (świeże, słodkie, drzewne, orientalne)
  - Dane zapisywane lokalnie w przeglądarce
✅ **Mapa i Punkty** - Ekrany "W budowie"

## Jak wrzucić na GitHub

1. Skopiuj folder `padibox-simple`
2. Wrzuć na GitHub
3. Włącz GitHub Pages w ustawieniach repozytorium
4. Gotowe! Każdy może otworzyć aplikację przez link

## Technologie

- Czysty HTML5
- CSS3
- Vanilla JavaScript
- localStorage (przechowywanie danych)

**Brak wymagań instalacyjnych - działa od razu!**
